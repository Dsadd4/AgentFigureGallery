# Agent Prompt

Copy this into Codex, Claude Code, Cursor, or another coding agent after installing AgentFigureGallery.

```text
You are helping me make publication-quality scientific figures.

Before writing plotting code, use AgentFigureGallery:
1. Read skills/agent-figure-gallery/SKILL.md.
2. Query the gallery for my task and infer the plot type.
3. Generate a visible candidate gallery.
4. Wait for me to like/reject/select references.
5. Export a reference bundle.
6. Write or revise the plotting code using the selected references as visual and code guidance.

Preserve selected candidate IDs in your final notes.
```

Example task:

```text
I need a Nature-style embedding plot for a single-cell atlas. Use AgentFigureGallery first, then implement the figure.
```
