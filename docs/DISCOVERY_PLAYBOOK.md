# Discovery Playbook

Goal: make AgentFigureGallery easy to find, understand, try, and share.

## Positioning

```text
Teach your plotting agent Nature, Cell, and Science figure taste in one minute.
```

## Share Copy

```text
I built AgentFigureGallery: a human-preference memory loop for scientific plotting agents.

Instead of asking an AI agent to invent a Nature/Cell/Science-style figure from text, it first looks at real visual references, lets the human like/reject/select, then writes code from the selected bundle.

16k+ public scientific figure candidates.
Dynamic browser gallery.
Global and plot-type preference memory.

GitHub: https://github.com/Dsadd4/AgentFigureGallery
```
